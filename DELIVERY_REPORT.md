### 核心内容

#### 1. 项目结构重构
```
accounting-app/
├── src/
│   ├── main/           # 主进程 (TypeScript)
│   ├── preload/        # Preload 脚本
│   ├── renderer/       # React 渲染进程
│   └── shared/         # 共享代码
└── tests/              # 测试代码
```

#### 2. 技术栈升级
- **语言**: JavaScript → TypeScript 5.3
- **前端**: 原生 JS → React 18 + Hooks
- **构建**: 无 → Vite 5
- **状态管理**: 无 → Zustand

#### 3. TypeScript 类型定义
- 核心类型定义 (Record, Category, Statistics)
- 验证函数 (validateRecord, validateCategory)
- API 服务层类型

#### 4. React 组件设计
- RecordForm (记账表单)
- RecordList (记录列表)
- RecordItem (记录项)
- 自定义 Hooks (useRecords, useCategories)

#### 5. 测试策略
- 单元测试 (Vitest + Testing Library)
- 集成测试
- E2E 测试 (Playwright)
- 目标覆盖率：>80%
